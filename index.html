What is Campfire?

This February, Hack Club & Open Sauce are running game jams in 200+ cities on the same weekend (including Bristol!).

About the Event

A game jam is an event where you build a game from scratch in a short time period! It's all about creativity, teamwork, and having fun while learning new skills.

Campfire Bristol brings together hackers, artists, and game enthusiasts in Bristol. Beginners and pros welcome!
